Help & Usage

Features

  • Named rule sets let you store multiple workflows.
  • Sync only when enabled by you.
  • Live group color preview and validation.
  • Auto-organize tabs on creation, URL changes, and new windows.
  • Quick popup actions for organize, reorder, clean, regroup, and ungroup.

Rule syntax

  • example.com — exact hostname or subdomain match.
  • *.amazonaws.com — wildcard hostname match.
  • contains:github — substring match anywhere in the hostname.
  • regex:^.*\.google\.com$ — regex match against the hostname.

Enter one rule per line. Save your rule set before switching to preserve changes.
